# Code of Conduct

This project aims to maintain a respectful, focused, and low-noise environment.

Participants are expected to:
- Engage in good faith
- Keep discussions technical and on-topic
- Respect differing viewpoints
- Avoid harassment, personal attacks, or disruptive behavior

The maintainer reserves the right to moderate, edit, or remove contributions
that are not aligned with these expectations.

This Code of Conduct applies to all project spaces, including issues, pull
requests, discussions, and other communication channels.
