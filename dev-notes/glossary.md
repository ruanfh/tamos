# Glossary

| Word          | Meaning                                                                     |
| ------------- | --------------------------------------------------------------------------- |
| Gem           | An object that complies with the normative schema (gem.submit.schema.json). |
| Star          | A Gem suitable for exploration.                                             |
| Sky           | The place where Stars are made available for exploration.                   |
| East-Horizon  | Receives a Gem and produces a Star within a Sky.                            |
| West-Horizon  | Permanently removes a Star from a Sky.                                      |
| TAMOS-Archive | Profile: append-only, immutable                                             |
| TAMOS-Field   | Profile: allows creation and removal of Stars, but not editing.             |
| TAMOS-Garden  | Profile: allows creation, removal, and editing of Stars.                    |
| Constellation | A group of Stars.                                                           |
| Scout         | An agent or process that discovers Gems or Stars for exploration.           |
| Navigator     | An agent or process that guides exploration within a Sky or among Stars.    |
| Harvester     | An agent or process that collects or aggregates Gems or Stars.              |
| Gardener      | An agent or process that manages, edits, or curates Stars in a Sky          |
| Offer         | The act of submitting a Gem for consideration or publication.               |
| Cast          | The act of creating or producing a Star from a Gem.                         |
| Harvest       | The act of collecting or removing Stars from a Sky.                         |
| Exchange      | The act of replacing one Star with another with the same identifier.        |
| Explore       | The act of discovering, inspecting, or interacting with Stars in a Sky.     |
